# Contributing

Thanks for your interest in contributing! This guide is intentionally short and easy to follow.

## Quick start

1. Fork this repo and clone your fork.
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Start the dev server:
   ```bash
   pnpm dev
   ```

## Branching

- Create your feature branch from `dev`.
- Use a clear name (example: `feature/improve-loading-text`).

## Commit messages

- Follow the Clean Commit format: `<emoji> <type>: <description>`.
- Example: `ðŸ“– docs: add contributing guide`.

## Pull requests

- Always open PRs against the `dev` branch.
- Keep PRs focused and small.
- Make sure lint passes:
  ```bash
  pnpm lint
  ```

## Code of conduct

By participating, you agree to follow the [Code of Conduct](CODE_OF_CONDUCT.md).
